# Blinkit Sales Analysis | SQL + Power BI

## About the Project

I created this project to analyze Blinkit's grocery sales data using Microsoft SQL Server and Power BI.

First, I cleaned and analyzed the data in SQL Server by writing queries to calculate business KPIs and understand sales performance. After completing the SQL analysis, I connected the cleaned data to Power BI and built an interactive dashboard to visualize the results.

## Tools Used

- Microsoft SQL Server
- SQL Server Management Studio
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Project Workflow

1. Imported and explored the dataset.
2. Cleaned and validated the data.
3. Performed sales analysis using SQL queries.
4. Calculated business KPIs.
5. Connected the data to Power BI.
6. Created an interactive dashboard with filters and visualizations.
7. Generated business insights from the dashboard.

## Business Requirements

The objective of this project is to analyze Blinkit's sales performance, customer satisfaction and product distribution using SQL and Power BI.

## KPI Requirements

- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Dashboard Analysis

The dashboard provides analysis of:

- Sales by Fat Content
- Sales by Item Type
- Sales by Outlet Size
- Sales by Outlet Location
- Sales by Outlet Establishment Year
- Sales by Outlet Type
- Average Sales
- Average Rating
- Number of Items

Interactive filters are available for:

- Outlet Location Type
- Outlet Size
- Item Type
